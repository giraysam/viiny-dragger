viiny-dragger

vinny-dragger is a drag and drop plugin for javascript.

View on GitHub Download Latest Release

Dragging

  • JS
  • HTML
  • CSS

var obj = document.getElementById('drag-1'),
    obj2 = document.getElementById('drag-2');
		
ViinyDragger(obj, {
  activeClass: 'active-drag'
});
	
ViinyDragger(obj2);
              

<div id="drag-1" class="draggable box">
    You can drag this
</div>

<div id="drag-2" class="draggable box">
    Drag me
</div>

	position: absolute;
	left: 0;
